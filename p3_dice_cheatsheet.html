P3 ニーズヘッグ「サイコロ」カンペ

サイコロ 1×3人 / 2×2人 / 3×3人 ・ 塔は数字順に出現 ・ 自分の塔は踏めない ・ 頭割りは常に5人＋軽減

ダイブデバフ = 塔が出る場所

ハイジャンプ真上 → 向き自由
スパイン前方 → ボスを向く
イルーシブ後方 → ボスに背を向ける

向きは着弾2秒前に固定。ボスをタゲって向き確認。

散開位置

ミックスイルーシブ=左(D側) / スパイン=右(B側) / HJ=真下
全員HJ優先度: 西 H1 MT ST D1 D2 D3 D4 H2 東
1・3番タゲサ淵の左下/真下/右下
2番タゲサから離れた北側 左=4-A間 / 右=A-1間

連旋(2回・毎回抽選)

牙尾 = 外→中 ／ 尾牙 = 中→外

頭割りはA側タゲサ淵(タゲサの外)。
①=2番+3番 ／ ②=1番+2番。
1番は頭割り①に入らないが内外回避は全員参加。

竜騎士の分身(直線)

塔を踏んだ本人が塔の外周側に立って外へ誘導。
対象決定は踏んだ約2秒後 → 向き確定を見てから離れる。

盤面 (①③=タゲサ淵 / ②=北側)

タイムライン

  1. 1番 左下/真下/右下で塔①設置(向き注意)。2・3番はA側淵で頭割り①
  2. 全員で連旋①回避(内外) → 1番はA正面へ(塔に残らない)
  3. 3番が塔①を踏む(自分が後で塔を出す場所と同じ)→直線誘導。2番は北で塔②設置(向き注意)→設置後すぐ頭割りへ
  4. 1番 左にいた人→左塔②・右にいた人→右塔②を踏み誘導→すぐ頭割り②へ。真ん中の1番は直行
  5. 連旋②回避 → 3番はその場(担当位置)で塔③設置(向き注意)→設置後A側へ
  6. 2番=左右の塔③、真ん中の1番=真ん中の塔③
    ルート: 尾牙=ボス下を中央突破 ／ 牙尾=円周沿い
    踏んだら直線誘導→終わったらA側集合

事故りやすいポイント

  • 塔を出したらその場を離れる(残る=踏めない塔を踏む)
  • ミックスの向きミス → 塔が変な場所に出て処理不能
  • 誘導後の離脱が早い → 直線が味方に向く(2秒待つ)
